Question 3

How does Flake differ from Pylint? The exercise was to run and use flake8 against several .py files and to assess the difference between flake and pylint. The obvious difference is that flake looks more at style errors (it will look at syntax as well) so would be better suited for identifying incorrect indentation and line spacing for example. For example on the pylint_test.py file flake8 returned an error on line 16 to indicate an error on whitespace around operators.
