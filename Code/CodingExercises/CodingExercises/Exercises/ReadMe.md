<div class="rendered-markdown"><h1>Running the toolkit application in Codio</h1>
<h3>Create a virtual machine</h3>
<p>Use **Tools > Install Software** and install X server.</p>
<h3>Configure X server</h3>
<p>Once X server is installed configure it to run the toolkit.py script</p>
<p>Use **Virtual Desktop > Configure** under the command node enter the following text - "Run Python (tkinter)": "python3 toolkit.py"</p>
<h3>Run your code</h3>
<p>Click **Run Python (tkinter)** menu to execute the toolkit.py script.</p>
<h3>Launch the User interface</h3>
<p>Click **Virtual Desktop** to execute the user interface.
<br  /><img src="resources/application.jpg" alt="" /></p>
<h3>Load JSON data</h3>
<p>Click the 'Load Released Toolkit Data' Button.</p>
<p>This will populate the Released Toolkits list box with all data that matches the status of 'Released'.
<br  /><img src="resources/loadeddata.jpg" alt="" /></p>
<p>The data in this list box is sorted on all records that are of Released status and also sorted/ordered by the numeric ID of each record in ascending order.</p>
<p>This satisfies the brief of sorting data.</p>
<h3>Displaying all data related to the items in the Released List</h3>
<p>Click the 'Read Released Toolkit Data' button.</p>
<p>If no item is selected in the released list then a messagebox will appear asking you to select data.
<br  /><img src="resources/selectreleasedmsg.jpg" alt="" /></p>
<p>This satisfies the brief of displaying a message to the user.</p>
<p>Select a record in the released list.</p>
<p>Click the 'Read Release Toolkit Data' button again. A message box will appear with all the data asscoated with that record.
<br  /><img src="resources/selectedmsg.jpg" alt="" /></p>
