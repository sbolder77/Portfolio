Assignments

The OOP module consisted of 3 assignments. This was for the system design and system implementation of a self driving car. The 3rd assignment was for the overall portfolio.

System design

This was the first assignment carrying a weighted mark for the overall module. It was to present a design proposal for a self driving car identifying 3 operations and presenting UML diagrams, proposed data structures and rationale for the selection of the 3 operations.

  • View System Design

System Implementation

The assignment was to deliver on the system design proposal by writing an Python program. Supporting documents such as UML diagrams, data structures etc. to be updated based on the evolution of design thinking.
