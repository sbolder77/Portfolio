Secure Software Development

Learning Outcomes

- Describe the key terms which are associated with testing software for security
- Prepare a list of questions to ask when designing a test plan for secure software
- Design software tests through understanding the range of ways which the security of software can be breached

Formative Activities - Coding, Seminars & Team Activities

The module covered a range of formative assignments icluding coding concepts and seminar activities to support the objectives and delivery of a team development assignment. The content covered a wide range of concepts all of which were developed through the Codio IDE or independently in Visual Studio. Most of the evaluation and prototyping was carried out in Visual Studio to utilise a more familiar development environment.

Unit 1

The activity was to write a discussive forum entry using a UML activity diagram and it's use to identify one of the OWASP top 10 weaknesses. Posts were reviewed by peers and included responses as well as a summary post.

Unit 2

The activity was to answer questions and discuss the software development stages of the scrum methodology. Additional activities were to discuss 5 items from the ISO/IEC 27000 standards and resource management in realtion to cyber security.

  • View Activity

Unit 3

The team were asked to work on a collaborative piece related on if a programming language is secure or not. We also performed some coding activities using Python.

  • View Activity

Unit 4

The seminar was based on a discussion around ReDOS and the use of 'Evil Regex'. We were asked to discuss the definition, common problems and mitigation with regex and how regex could be used in a secure solution

Unit 5

This coding activity was to execute and analyse an example of equivalence in Python. We also had to write a piece on the relevance of Cyclomatic Complexity.

Unit 6

The activity was to execute and investigate the use of linters in Python for the purposes of supporting testing. We had to execute several pieces of code utilising different linter libraries and provide an assessment piece.

Unit 7
